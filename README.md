# Binary-Decimal Converter

Simple C++ application to convert between `Binary` and `Decimal` bases.

## Purpose

This application built for practical purposes.

## Who can use it?

Anyone.

## How to use?

 * `$ cd build`
 * `$ cmake ..`
 * `$ make`
 * `$ ./main` or `$ ./test_1` or `$ ./test_2`
